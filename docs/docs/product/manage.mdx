# Manage Product

Kamu dapat mengatur produk, mulai dari mengatur data produk seperti harga, stok, jenis produk

## Product Data

Merupakan kumpulan data yang dibutuhkan sebuah produk dapat dibeli. dan untuk pengkategorian dari jenis produk.

### Harga

- Harga Normal
- Harga Discount

### Stok

- Stock (int) 4
- Stock Unit (string) Pcs
- Max Purchase (int) -1 // Unlimited
- Min Purchase (int) 1

### Jenis Produk

terdapat 2 buah tipe produk 

- Digital 
  - File url
  - Versi
- Fisik
  - Berat
  - Volume

### Extendable

Kamu bisa menambah tab dan bidang dari produk manage menggunakan hook 
`do_action("lwpcommerce/product/manage/fields");`
