# Notification API

Payment API berguna ketika anda ingin membuat tampilan metode pembayaran mana yang aktif.
atau juga ingin menampilkan detail pembayaran, untuk perhitungan total di Frontend.

## Endpoints

### <code class="method-request-get">GET</code> `/lokuswp/v1/payment/list`

Endpoint yang berguna untuk menampilkan daftar pembayaran yang aktif
untuk pengaturan pembayaran aktif akan diatur di halaman admin wordpress.

#### Code samples
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
    groupId="codeGroup1"
    defaultValue="shell"
    values={[
    {label: 'Shell', value: 'shell'},
    {label: 'Javascript (fetch)', value: 'javascript(fetch)'},
    {label: 'Axios', value: 'axios'},
    {label: 'PHP', value: 'php'},
]}>

  <TabItem value="shell">

  ```shell
  curl --location --request GET 'https://beta.lokuswp.com/lwpcommerce/wp-json/lokuswp/v1/payment/list'
  ```

  </TabItem>

  <TabItem value="javascript(fetch)">

  ```javascript
  var requestOptions = {
    method: 'GET',
    redirect: 'follow'
  };

  fetch("https://beta.lokuswp.com/lwpcommerce/wp-json/lokuswp/v1/payment/list", requestOptions)
    .then(response => response.text())
    .then(result => console.log(result))
    .catch(error => console.log('error', error));
  ```

  </TabItem>

  <TabItem value="axios">

  ```javascript
  var axios = require('axios');

  var config = {
      method: 'get',
      url: 'https://beta.lokuswp.com/lwpcommerce/wp-json/lokuswp/v1/payment/list'
  };

  axios(config)
      .then(function (response) {
          console.log(JSON.stringify(response.data));
      })
      .catch(function (error) {
          console.log(error);
      });

  ```

  </TabItem>

  <TabItem value="php">

  ```php
  $curl = curl_init();

  curl_setopt_array($curl, array(
    CURLOPT_URL => 'https://beta.lokuswp.com/lwpcommerce/wp-json/lokuswp/v1/payment/list',
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_ENCODING => '',
    CURLOPT_MAXREDIRS => 10,
    CURLOPT_TIMEOUT => 0,
    CURLOPT_FOLLOWLOCATION => true,
    CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
    CURLOPT_CUSTOMREQUEST => 'GET',
  ));

  $response = curl_exec($curl);

  curl_close($curl);
  echo $response;
  ```

  </TabItem>

</Tabs>
